# App Flutter

Projeto de estudo de Flutter, com o objetivo de criar um aplicativo para o sistema Android e IOS.

## Descrição

O aplicativo não tem um foco especifico, apenas estudo de Flutter. Criado na disciplina de Desenvolvimento de Aplicativos Móveis.

## Instalação

Para instalar o projeto, basta clonar o repositório e executar o comando abaixo:

```bash
flutter pub get
```

## Execução

Para executar o projeto, basta executar o comando abaixo:

```bash
flutter run
```

## Ferramentas

- [Flutter](https://flutter.dev/)
- [Dart](https://dart.dev/)
- [Android Studio](https://developer.android.com/studio)
- [Visual Studio Code](https://code.visualstudio.com/)
